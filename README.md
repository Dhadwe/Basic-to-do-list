This project is a basic to do list made using python

this to do list uses pYQT6 to have a simple and user friendly interface.
The to do list also follows a easy on the eyes and warm theme


Ai was used for the stylesheets


The user can add tasks to the list easily
and delete them when done
acting as a simple and basic to do list
with a nice looking interface
